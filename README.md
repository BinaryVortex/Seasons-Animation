# Seasons Animation

A beautiful, animated representation of the four seasons using HTML5 Canvas, CSS, and JavaScript. The animation automatically cycles through Spring, Summer, Autumn, and Winter, showcasing seasonal elements like flowers, beaches, falling leaves, and snowscapes.

## 🌐 Live Demo

View the live animation: [https://binaryvortex.github.io/Seasons-Animation/](https://binaryvortex.github.io/Seasons-Animation/)

## ✨ Features

- **Automatic Cycling**: Seasons change every 3 seconds automatically
- **Responsive Design**: Adapts to different screen sizes
- **Smooth Animations**: Canvas-based rendering for fluid transitions
- **Seasonal Themes**:
  - **Spring**: Blossoming flowers, butterflies, and green landscapes
  - **Summer**: Sunny beaches with waves, umbrellas, and beach balls
  - **Autumn**: Colorful foliage with falling leaves and orange skies
  - **Winter**: Snow-covered scenes with snowmen and snowflakes

## 🛠️ Technologies Used

- **HTML5**: Structure and canvas element
- **CSS3**: Styling and responsive layout
- **JavaScript**: Animation logic and canvas drawing

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Seasons-Animation.git
   ```

2. Open `index.html` in your web browser.

That's it! The animation will start automatically.
